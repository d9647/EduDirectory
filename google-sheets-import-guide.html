Google Sheets to CSV Import Guide

Educational Yellow Pages Platform

Step-by-step guide for importing data from Google Sheets

Step 1: Prepare Your Google Sheet

Create a new Google Sheet or open an existing one with your data.

Important: Make sure your first row contains the exact column headers as shown in the CSV template. Column order doesn't matter, but the names must match exactly.

Required Column Headers (copy and paste these exactly):

For Tutoring Providers:

name,type,description,categories,subjects,city,state,deliveryMode,website,phone,email,address,photoUrl

For Summer Camps:

name,description,categories,city,state,cost,selectivityLevel,minimumAge,hasScholarship,tags,dates,length,applicationOpen,applicationDeadline,deliveryMode,website,phone,email,address,photoUrl

For Internships:

companyName,title,description,types,city,state,compensation,duration,minimumAge,hasMentorship,deliveryMode,applicationOpen,applicationDeadline,internshipDates,prerequisites,tuition,website,phone,email,address,photoUrl

For Jobs:

companyName,title,description,categories,city,state,jobType,schedule,minimumAge,hasTraining,salaryRange,salaryType,deliveryMode,openingDate,closingDate,isOngoing,website,phone,email,address,photoUrl

Step 2: Format Your Data

Array Fields (Multiple Values):

For fields that accept multiple values, separate them with semicolons (;):

  • Categories: STEM;Technology;Science
  • Subjects: Mathematics;Physics;Chemistry
  • Tags: Advanced;Competitive;Summer
  • Types: Technology;Software;Engineering
  • Duration: Summer;Full-time;Academic Year
  • Job Type: Part-time;Full-time
  • Schedule: Day shift;Monday to Friday;Weekends

Date Fields:

Use the format YYYY-MM-DD for all dates:

  • Good: 2024-06-15
  • Bad: 6/15/2024 or June 15, 2024

Boolean Fields:

Use any of these values for true/false fields:

  • True: true, TRUE, 1, yes, YES
  • False: false, FALSE, 0, no, NO
  • Empty: Leave blank for undefined/optional boolean fields

Cost Field (Summer Camps Only):

Use these exact values for summer camp costs:

  • < $1000
  • $1000-$3000
  • $3000-$5000
  • $5000-$6500
  • > $6500

Step 3: Download as CSV

  1. In your Google Sheet, click File in the top menu
  2. Select Download
  3. Choose Comma Separated Values (.csv)
  4. Save the file to your computer
[Screenshot would show: File → Download → Comma Separated Values (.csv)]
Warning: Make sure to download as CSV, not Excel or other formats. The import system only accepts CSV files.

Step 4: Upload to Admin Panel

  1. Log in to the admin panel
  2. Navigate to the Import Data tab
  3. Select the appropriate import section (Tutoring Providers, Summer Camps, Internships, or Jobs)
  4. Click Choose File and select your CSV file
  5. Click the Import button
Note: The import process will show you how many records were successfully imported and any errors that occurred.

Step 5: Review Import Results

After importing, you'll see a summary showing:

  • Success Count: Number of records successfully imported
  • Error List: Detailed errors for any failed imports
Success! All imported items will appear in the "Pending Approvals" tab and need to be reviewed and approved before they become live on the platform.

Common Issues and Solutions

❌ "Missing required fields" error

Solution: Ensure all required fields (name, description, city, state, categories) have values in every row.

❌ "CSV parsing error"

Solution: Make sure your file is saved as CSV format and contains valid data without special characters in the wrong places.

❌ "Invalid date format"

Solution: Use YYYY-MM-DD format for all date fields.

❌ Rows being skipped

Solution: Check that all required fields are filled and that column headers match exactly.

❌ Array fields not working

Solution: Use semicolons (;) to separate multiple values, not commas or other separators.

Tips for Large Imports

  • Test Small Batches: Start with 5-10 records to test your format before importing hundreds
  • File Size Limit: Keep CSV files under 10MB for optimal performance
  • Prepare Data Offline: Clean and format your data completely before uploading
  • Backup Your Data: Keep a copy of your Google Sheet as backup
  • Review Templates: Always reference the CSV templates for the exact format required

Need Help?

If you're still having trouble with imports:

  1. Download and review the CSV templates
  2. Check that your column headers match exactly
  3. Verify your data formatting (especially dates and boolean values)
  4. Try importing a small test batch first
  5. Contact the platform administrator if issues persist
