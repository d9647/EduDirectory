Educational Yellow Pages - Google Sheets Import Template

This template provides the exact column headers needed for bulk importing data into the Educational Yellow Pages platform. Copy the column headers for your specific business type and create your Google Sheet accordingly.

How to Use This Template:

  1. Choose your business type from the sections below
  2. Copy the column headers for that type
  3. Create a new Google Sheet with these exact column names as the first row
  4. Fill in your data below the headers
  5. Export as CSV or share with the platform administrators

Important: Column names must match exactly (case-sensitive). Required fields are highlighted in red.

General Import Guidelines

Data Format Requirements:

  • Required fields (*) must have values for each row
  • Array fields (blue) can contain multiple values separated by commas
  • Boolean fields: Use TRUE or FALSE (case-insensitive)
  • Date fields: Use YYYY-MM-DD format (e.g., 2025-06-15)
  • Numeric fields: Use decimal notation (e.g., 15.50, not $15.50)
  • URLs: Include full URL with https:// or http://
  • Empty cells: Leave blank for optional fields
  • Special characters: Avoid using commas in single-value fields (use semicolons instead)
