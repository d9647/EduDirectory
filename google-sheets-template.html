Important Setup Instructions

  • Create 4 separate sheets in your Google Sheets document with these exact names: "Tutoring Providers", "Summer Camps", "Internships", "Jobs"
  • Copy the column headers from each table below into the first row of each corresponding sheet
  • Required fields (red background) must have values - leave others blank if not applicable
  • Array fields (blue background) should contain multiple values separated by commas
  • Boolean fields (green background) should contain TRUE or FALSE
  • Date fields (orange background) should use YYYY-MM-DD format (e.g., 2024-07-15)

After Creating Your Spreadsheet

  1. Fill in your data following the examples and field requirements above
  2. Make sure required fields (red background) are not empty
  3. For array fields (blue background), separate multiple values with commas
  4. Use TRUE/FALSE for boolean fields (green background)
  5. Use YYYY-MM-DD format for dates (orange background)
  6. Save your Google Sheets and share it with view access if needed
  7. Export individual sheets as CSV files for import into the website
