Important Notes:

  • Array Fields: Use semicolons (;) to separate multiple values in array fields
  • Date Format: Use YYYY-MM-DD format for all date fields
  • Boolean Fields: Use "true", "false", "1", "0", "yes", or "no" for boolean fields
  • Required Fields: All required fields must have values, or the row will be skipped
  • URLs: Include the full URL including http:// or https://
  • File Format: Save as CSV (UTF-8) format
  • No Headers: Make sure the first row contains the exact column names as shown above
  • Approval Process: All imported items will be pending approval and need to be approved by an admin
